# القرآن الكريم (Quran App)

تطبيق بسيط ومفتوح المصدر لقراءة القرآن الكريم، مصمم لتوفير تجربة قراءة مريحة وهادئة.

## المميزات (Features)

*   **تجربة قراءة متميزة:** تصميم احترافي مستوحى من ألوان المصحف الشريف (الأخضر والذهبي) مع خلفية ورق مريحة للعين.
*   **التمرير التلقائي (Auto-Scroll):** ميزة التمرير التلقائي مع إمكانية ضبط السرعة بدقة (15 مستوى) لتناسب سرعة قراءتك.
*   **نظام العلامات المرجعية:** حفظ العلامات المرجعية، إمكانية إعادة تسميتها، وحذفها للرجوع إلى مكان توقفك بسهولة.
*   **متابعة الختمة:** تتبع آخر زيارة وتنبيهك بمكان توقفك (آخر سورة وآية).
*   **البحث السريع:** إمكانية البحث عن أي آية في القرآن الكريم والوصول إليها فوراً.
*   **الانتقال السريع:** واجهة سهلة للانتقال إلى أي سورة أو آية محددة.
*   **وضع عدم الإزعاج التلقائي:** تفعيل وضع "عدم الإزعاج" تلقائياً أثناء القراءة لتجنب المقاطعات.
*   **تنبيهات الأجزاء والأحزاب:** إشعارات عند الوصول إلى ربع حزب أو بداية حزب جديد.
*   **منع إغلاق الشاشة:** تبقى الشاشة مفعلة طوال فترة القراءة.

## التقنيات المستخدمة (Tech Stack)

*   **Kotlin:** لغة البرمجة الأساسية.
*   **Jetpack Compose:** لبناء واجهات المستخدم الحديثة والديناميكية.
*   **Material 3:** تطبيق أحدث معايير التصميم من جوجل.
*   **Gson:** للتعامل مع البيانات بصيغة JSON.
*   **Coroutines:** لإدارة المهام الخلفية وسلاسة التمرير.

## المتطلبات (Requirements)

*   أندرويد 7.0 (API Level 24) أو أحدث.
